# Chrome Extension

Load this folder as an unpacked extension in Chrome:

1. Open `chrome://extensions`
2. Enable Developer Mode
3. Click `Load unpacked`
4. Select the `chrome-extension` folder

Use the extension by selecting text on a page, right-clicking, and choosing `Translate to Darija`.

Default backend URL:

`http://localhost:8080/darija-translator/api/translate`

Default demo credentials:

- Username: `demo`
- Password: `demo123`

The side panel includes:

- Selected text capture
- Translation request
- Copy result
- Read aloud
- Optional voice input
